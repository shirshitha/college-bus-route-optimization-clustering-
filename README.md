# college-bus-route-optimization-clustering-
College Bus Route Optimization Using Clustering
Project Overview
This project focuses on optimizing college bus routes using Machine Learning techniques. The system uses the K-Means Clustering Algorithm to group students based on their residential locations (latitude and longitude).
By clustering nearby students together, the college transportation department can design efficient bus routes that:
Reduce fuel consumption
Minimize travel distance
Save transportation costs
Improve student convenience
This project demonstrates how clustering can be applied in real-world transportation systems.
Features
Student location clustering

Efficient transportation planning
Scalable for larger datasets

Dataset
The dataset contains student residential coordinates such as latitude and longitude values. These values are used to group nearby students into clusters for optimized route planning.
Working of the System
Collect student location data.
Apply K-Means clustering algorithm.
Group nearby students into clusters.
Identify optimized bus stop centers.
Generate efficient transportation routes.
Algorithm Used — K-Means Clustering
K-Means is an unsupervised machine learning algorithm used to divide data into clusters based on similarity.
Steps Involved
Select the number of clusters.
Initialize cluster centroids.
Assign data points to nearest centroid.
Recalculate centroid positions.
Repeat until clusters stabilize.
Output
The system groups nearby students into clusters and helps in designing optimized college bus routes that reduce travel time and transportation cost.
